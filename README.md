# 📖 معرفی درس

سیستم عامل یکی از بنیادی‌ترین دروس رشته مهندسی کامپیوتر است که نقش واسط میان سخت‌افزار و نرم‌افزار را بر عهده دارد. درک صحیح مفاهیم این درس، پیش‌نیاز بسیاری از مباحث پیشرفته مانند سیستم‌های توزیع‌شده، رایانش ابری، امنیت سیستم، سیستم‌های نهفته، مجازی‌سازی، سیستم‌های بلادرنگ و طراحی سیستم‌های عامل است.

در این Repository تلاش شده است علاوه بر ارائه مباحث نظری، ارتباط مفاهیم با مسائل واقعی، نمونه‌های عملی، پروژه‌ها و فعالیت‌های آموزشی نیز پوشش داده شود تا دانشجویان بتوانند دید عمیق‌تری نسبت به عملکرد سیستم‌های عامل پیدا کنند.

---

# 🎯 اهداف آموزشی

در پایان این درس انتظار می‌رود دانشجویان بتوانند:

* مفهوم و وظایف سیستم عامل را به‌طور دقیق تشریح کنند.
* ساختارهای مختلف سیستم‌های عامل را تحلیل و مقایسه نمایند.
* نحوه ایجاد، مدیریت و زمان‌بندی پردازه‌ها را درک کنند.
* الگوریتم‌های مختلف زمان‌بندی CPU را تحلیل و ارزیابی کنند.
* مفاهیم همزمانی، نخ‌ها و همگام‌سازی را به‌کار گیرند.
* مسائل بن‌بست را تحلیل کرده و روش‌های پیشگیری، اجتناب و بازیابی از آن را توضیح دهند.
* روش‌های مدیریت حافظه و حافظه مجازی را بررسی کنند.
* اصول مدیریت فایل، دیسک و سیستم‌های ذخیره‌سازی را تحلیل نمایند.
* مفاهیم حفاظت، امنیت و کنترل دسترسی در سیستم‌های عامل را توضیح دهند.
* دانش کسب‌شده را در تحلیل مسائل عملی و طراحی راهکارهای مناسب به‌کار گیرند.
